&gt; **AI-Powered Resume Analysis & Career Growth Platform**

CultivationX helps developers analyze their resumes, identify skill gaps, track coding progress (GitHub + LeetCode), and get AI-powered mentorship — all in one place.

---

## 🚀 Features

| Module | Description |
|--------|-------------|
| **🔥 RISE** | AI resume parser + skill gap analysis |
| **🧬 DevDNA** | AI-powered developer personality report |
| **🌐 Nexus** | GitHub & LeetCode profile sync |
| **👨‍🏫 Mentor** | AI coding mentor with code review |
| **📊 Dashboard** | Unified progress tracking |

---

## 🛠️ Tech Stack

### Backend
- **Java 24** + **Spring Boot 3.5**
- **Spring AI** (Groq API + Google Gemini)
- **Spring Security** + JWT Authentication
- **Spring Data JPA** + MySQL
- **Apache Tika** (PDF/DOCX parsing)

### Frontend
- **React 18** + **TypeScript**
- **Vite** (build tool)
- **CSS Modules**

### DevOps
- **Docker** + **Docker Compose**
- **GitHub Actions** (CI/CD ready)
- **Nginx** (reverse proxy)
